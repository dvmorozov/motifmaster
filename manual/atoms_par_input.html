Table of atom properties

X, Y, Z - coordinates of atom in the basis of unit cell.

Mx, My, Mz - direction of magnetic moment (normalized vector) in the basis of unit cell. The vector is normalized automatically. Simply fill desired direction as, for example, Mx = 1, My = 1, Mz = 1.

Module - value of the magnetic moment - "module".

NSA - nuclear scattering amplitude of the atom.

Element - symbol of chemical element of the atom. This symbol can be arbitrary, but must coincide with the symbol of desired element in the form factors database. If current site is not magnetic (i.e. having no magnetic propagation vectors) this symbol is ignored.

Site - name of the current site. This symbol can be arbitrary.

Ell. Param. - start value of ellipse parameter of ES type structure. For other types of structures is ignored.

Note that before input of atom parameters the unit cell parameters should be set properly. Otherwise Mx, My, Mz will be filled by zeros.
