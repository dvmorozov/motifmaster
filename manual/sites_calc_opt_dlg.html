Site calculation options

Site is taking part - turning off excludes this site from calculation at all.

Site is magnetic - turning off disables generating the magnetic intensities from this site.

Representations Using Modes have following meanings:

By One Representation - in this mode the basal functions from only single representation take part in forming the magnetic structure on the atoms of site. All possible combinations of representations are selected one by the other.

By All Representations - all basal functions take part simultaneously.

Don't Use The Representations - this mode disables using the basal functions at all and activates Variation Mode selector.

Variation Modes have following meanings:

Vary All Of Moment Components - in this mode all components and "modules" of magnetic moment varies simultaneously and independently. This mode doesn't put any restrictions on possible magnetic structures. Using this mode only for small number of magnetic atoms or (and) very accurate experimental data is recommended.

Save Equal Modules At The Site - in this mode the directions of magnetic moments vary independently, but all moments have the same "module".

Only Modules Vary - in this mode all moment directions are fixed.
